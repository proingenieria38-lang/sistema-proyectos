Carta Gantt del proyecto

Carta Gantt guardada
La carta Gantt queda asociada al proyecto que elijas. El nombre de cada tarea y sus fechas se ingresan manualmente, de forma independiente a la tabla del presupuesto.
Selecciona un proyecto de la lista para crear o ver su carta Gantt.

Dashboard de avance

Filtra por proyecto o deja "Todos los proyectos" para ver el resumen general. Los datos se toman directamente de la Carta Gantt de cada proyecto.
Todavía no hay tareas de Carta Gantt guardadas.

PDF por proyecto

Sube, visualiza, imprime o elimina los archivos PDF de este proyecto (cotizaciones, contratos, respaldos, etc.). Se guardan en este navegador, así que evita archivos muy pesados.
Selecciona un proyecto de la lista para gestionar sus PDF.
